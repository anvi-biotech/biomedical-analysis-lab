# Biomedical Analysis Lab - Full-Stack DNA Visualizer

A full-stack web application for rapid DNA sequence processing, quality control, and codon detection.

## 🧬 Features
* **Sequence Cleaning:** Cleans raw DNA sequences and checks sequence length.
* **Codon Detection:** Identifies start (`ATG`) and stop (`TAA`) codons.
* **Corruption Check:** Ensures no invalid characters or numbers are in the sequence.
* **Modern UI:** Built with an interactive dark-mode interface powered by a Python (Flask) backend.

## 🛠️ Tech Stack
* **Backend:** Python, Flask
* **Frontend:** HTML5, CSS3, JavaScript
